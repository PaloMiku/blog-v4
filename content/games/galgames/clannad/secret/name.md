::alert

此特性仅限可修改主角名字的 Full Voice 旧版。

::

在 Clannad 中，玩家可以对主角的姓名进行修改（原名为：岡崎 朋也），而修改名字这一功能不仅能够在一定程度上加强玩家的代入感，有一些十分特殊的名字还会在一些特殊情况下令游戏的台词发生变化。

以下就将这些名字和对应的变化列举出来：

## 日文版

| 日期   | 场景 | 条件 | 示例 | 备注 |
|--------|------|------|------|------|
| **智代路线** ||| | |
| 4.17 | 春原第二次挑战智代的时候。<br>若发生春原脸部变形事件则不发生此事件。 | 名字不是“朋也” | 岡崎 岡崎 | 朋也和智代的名字的首字发音都是“TOMO”，春原发出了“TOMO组合”的感叹，改名后自然不再成立。 |
| **有紀寧路线** | | | | |
| 4.18 | 有紀寧记下主角的姓名的时候。以及之后几次不良们登场时。 | 姓“蛭子” | 蛭子 朋也 | 未改名时有紀寧误记为“蛭子”，改了之后误记为“江の川”。 |
| **渚路线** | | | | |
| 4.16 | 与古河一家吃饭时。 | 情况1：姓“銀河”或“宇宙”或“コスモ”，名“シャア”<br><br>情况2：姓“銀河”或“宇宙”或“コスモ”，名为“シャア”以外的任意名字 | 情况1：銀河　シャア<br><br>情况2：銀河　朋也 | 未改名时，古河一家在调侃朋也的名字没有气势，并提到了这些特别的词语。改名后，自然就会改变台词了。 |
| 4.17 | 秋生和早苗晚上来公园接渚的时候。 | 情况1：姓名为“コスモ　斉藤”<br><br>情况2：姓名为“大宇宙　太郎”<br><br>情况3：姓名中包含了“コスモ”或“銀河”或“宇宙”，但与情况1和2不同 | 情况1：コスモ　斉藤<br><br>情况2：大宇宙　太郎<br><br>情况3：銀河　朋也 |  |
| 4.19 | 在面包店与秋生见面时。 | 情况1：姓“銀河”或“宇宙”或“コスモ”<br><br>情况2：名“銀河”或“宇宙”或“コスモ”<br><br>情况3：不姓“銀河”或“宇宙”或“コスモ”且名为“ウジ虫” | 情况1：銀河　朋也<br><br>情况2：岡崎　銀河<br><br>情况3：岡崎　ウジ虫 |  |
| 4.19 | 去找渚时被早苗叫住。 | 姓“宇宙” | 宇宙　朋也 |  |
| 5.2 | 得知早苗开了学社时。 | 姓或名为“歌丸” | 岡崎　歌丸 |  |
